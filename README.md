<div align="center">Preview extractor for .2dx files (Addition to - https://github.com/Glebsin/iidx2bms) </div>




# HOW TO USE

1. Download [`2dx_preview_extractor.zip`](https://github.com/Glebsin/2dx_preview_extractor/releases/download/2025.130.0/2dx_preview_extractor.zip) from Releases and unzip `2dx_preview_extractor` folder
2. Open `2dx_preview_extractor.exe`
3. Follow instructions in console
4. Get result in `output` folder
